# Blake-Snell-Playoff-Pitch-by-Pitch-Analysis
This project uses Markov chain modeling to analyze the pitch sequencing tendencies of MLB pitcher Blake Snell during the 2025 postseason. Using pitch-by-pitch data from all 537 pitches thrown across six playoff appearances, the analysis estimates the probability of each pitch type being thrown based on previous pitches within a plate appearance.

The resulting Markov chain model provides insight into the predictability of Snell's pitch arsenal by estimating transition probabilities between pitch types and evaluating how previous pitches influence future pitch selection. This project demonstrates the application of probabilistic processes and Markov chains in sports analytics.

The repository contains the Python code used to construct the Markov chain model, along with the Excel dataset containing pitch-by-pitch data from Blake Snell's 2025 playoff performances.
